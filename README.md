# frederikschnack.github.io

[![Build and deploy](https://github.com/FrederikSchnack/frederikschnack.github.io/actions/workflows/hugo.yaml/badge.svg?branch=main)](https://github.com/FrederikSchnack/frederikschnack.github.io/actions/workflows/hugo.yaml)

This is the repository for my [personal homepage](https://frederikschnack.github.io/) using [hugo](https://github.com/gohugoio/hugo) with the [PaperMod](github.com/adityatelange/hugo-PaperMod) theme. 

You can start a local webserver by running
`
hugo server
`.
